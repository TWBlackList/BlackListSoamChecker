# ReimuBot Plugin: BlackListSoamChecker

## How to build ReimuBot Plugin: BlackListSoamChecker

1. [Install .Net Core 2.0](https://www.microsoft.com/net/core)

2. [Build ReimuAPI.dll](https://github.com/cnBlackList/ReimuAPI/blob/master/README.md)

3. Get source: `git clone https://github.com/cnBlackList/BlackListSoamChecker`

4. Put ReimuAPI.dll to `.`

5. Run `dotnet build`
